# 毕业论文目录：基于Qwen3-VL的校园网违规图片智能审查系统与轻量级部署研究

# 第1章 绪论
## 1.1 研究背景
### 1.1.1 校园网络内容安全面临的严峻挑战
阐述校园网上传图片数量激增与违规内容（色情、暴恐、政治敏感及隐喻图片）隐蔽化之间的矛盾。
### 1.1.2 现有图像审核技术的瓶颈分析
传统重型CNN的局限：虽然识别速度快，但缺乏语义理解能力，难以识别隐喻、变体及对抗样本，且误报率较高。
单一VLM模型的部署困境：虽然具备强大的多模态理解能力，但参数量大、推理延迟高、算力成本昂贵，难以直接应用于校园网边缘侧或低成本服务器的高并发场景。
### 1.1.3 “轻量级CNN + VLM”级联架构的提出
引出本研究的核心思路：利用轻量级CNN（如改进版YOLO）进行高频、快速的初步筛查与定位，仅将疑似或复杂样本交由VLM进行深度语义研判，以平衡效率与精度。
## 1.2 研究目的
### 1.2.1 突破单一模型在隐喻与复杂场景下的识别困境
利用VLM的多模态推理能力，解决传统CNN无法理解图片中文字梗、隐晦符号及上下文关联的问题。
### 1.2.2 构建基于轻量级CNN的高效前置过滤机制
设计并优化适用于校园网场景的轻量级CNN模型，实现对常规违规图片的毫秒级拦截，大幅降低进入VLM处理流程的数据量。
### 1.2.3 验证多模态协同架构在边缘侧的低成本部署可行性
探索“轻量级CNN（端侧/边缘侧）+ 量化VLM（服务器侧）”的协同部署方案，评估其在低算力硬件上的资源占用、响应时间及并发处理能力。
## 1.3 国内外研究现状
### 1.3.1 基于深度学习的图像内容审核技术研究
综述从传统机器学习到深度CNN（ResNet, EfficientNet等）在违规图片检测中的应用及局限性。
### 1.3.2 轻量级目标检测与分类网络的研究进展
重点综述：YOLO系列（v5/v8/v10等）、MobileNet、ShuffleNet等轻量级模型在图像快速识别领域的优化策略（剪枝、蒸馏、量化）及其在内容安全领域的最新应用。
### 1.3.3 视觉语言模型（VLM）在安全领域的应用与优化
分析LLaVA、Qwen-VL等多模态大模型在理解复杂语义方面的优势，以及当前针对VLM的轻量化（LoRA微调、量化压缩）研究现状。
### 1.3.4 多模型级联与协同推理架构研究综述
总结“粗筛 + 精判”的双阶段或多阶段检测框架在工业界的应用案例，分析其在平衡准确率与推理速度方面的技术路线。
